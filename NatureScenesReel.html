3D Generators

3D Generators are made by Houdini,which is a procedural tool and users can use it to build their custom tools.

Tree and Shrub Generator

Modeling

This generator can create trees started from the trunk, and you can modify the details of each layer of branches. And for Beginners, there is a random seed to generate trees automatically.

Most of the UV pieces are overlapped, I reused a lot of the space of the UV map, although this will cause more seams, this allows us to reuse textures for different 3D models. Only when we change the tree style will we need to rebake maps.

Rock Generator

The rock generator can create stones started from a basic shape, such as a cube, and you can also start from any shape in the FBX file.

  • Seed Polygon
  • High-poly Mesh
  • Low-poly Mesh

Rock Generator in Houdini

the rock generator can only work in Houdini because the high polygon generation and rendering will consume most of the performance in Unity

2D Generators

  • All of the Texture Generators are designed both for beginners and masters.
  • Texture generators are working in Substance Designer.
  • For Rocks, it relies more on baked normal maps, so it will be procedurally generated. But for plants, it relies less on baked normal maps, I used some leaf textures from texture.com.

Shaders

Shaders for Grasses and Trees

I select the billboard shader to reduce polygons because, in this shader, the surface will be always rendered face to camera, which means fewer polygons are required for each grass.

Both the tree and grass can be pushed by the wind. For grasses, they move with the UV offset, while for trees, they move with the offset of vertex position.
